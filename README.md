# **Treasure Hunt**  

**Treasure Hunt** — это увлекательный 2D-платформер, в котором игрок управляет отважным искателем сокровищ.  

## **Особенности игры**  
✅ **Динамичное управление:**  
- **← →** — движение влево/вправо  
- **↑ / Пробел** — прыжок  
- **Двойной прыжок** — дополнительный толчок в воздухе  

🎯 **Цель:**  
- Добраться до **сундука с сокровищами**, чтобы перейти на следующий уровень.  
- Собирать **ключи** (1-5 на уровень), чтобы открыть сундук.  
- Избегать **лезвий** (при касании — поражение).  

⚡ **Сложность и прогрессия:**  
- Уровни становятся сложнее: больше ловушек.  


🖌️ **Стиль:**  
- Яркая пиксельная/мультяшная графика.  
- Плавная анимация и отзывчивое управление.  

## **Требования**  
- **C++17** (или новее)  
- **CMake** (для сборки)  
- **FLTK 1.4+** (или другая графическая библиотека)  

## **Сборка и запуск**  
```bash
cmake -B build
cmake --build build
./build/TreasureHunt  # или TreasureHunt.exe на Windows
```
